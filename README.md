# Wpr381 — Eventurer

A full-stack event booking platform built with Node.js, Express, EJS, and MongoDB.

## Setup

1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Open `http://127.0.0.1:6969` in browser

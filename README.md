# WPR381 Project: Eventurer

A full-stack event booking platform built with Node.js server-side runtime, Express.js web app framework, EJS, HTML5, CSS3, and MongoDB + Mongoose ODM.

## Setup

### Prerequisites

#### **[Node.js (LTS)](https://nodejs.org/en/download)**

#### **npm**

If you install Node from the link above, you already have npm.

#### **[MongoDB Community Server](https://www.mongodb.com/try/download/community)**

MongoDB will run locally on port 27017. It is recommended to install MongoD as a Service (Network Service) via the wizard unless you know what you are doing.

#### **[Git for Windows](https://git-scm.com/download/win)**

To clone repository.

## Installation

1. Clone the repo
   1. Run `git clone https://github.com/Bodebate/Wpr381.git`
   2. Run `cd Wpr381`
2. Install dependiences:
   1. Run `npm install`
3. (SKIP if you have MongoD as a Service) Start the database:
   1. Run `mongod`
4. Seed database (will reset data):
   1. Run `npm run seed`
5. Start server:
   1. Run `npm start`
6. Open [http://127.0.0.1:6969](http://127.0.0.1:6969) in browser
